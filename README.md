# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how trader performance changes across different market sentiment regimes.

## Data
- Trader execution data
- Daily market sentiment classification

## Key Findings
- Extreme Greed shows highest mean PnL and win rate.
- Fear regimes increase volatility and reduce win rate.
- High-size traders outperform low-size traders.
- Overtrading reduces profitability during volatile regimes.

## Strategy Recommendations
1. Increase allocation during Extreme Greed.
2. Reduce exposure during Extreme Fear.
3. Avoid high frequency trading in volatile periods.

## Tools Used

- Pandas
